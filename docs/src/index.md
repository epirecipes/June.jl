# June.jl

`June.jl` is a Julia port of [JUNE](https://github.com/IDAS-Durham/JUNE), the individual-based epidemiology simulator.

This docs site publishes the package tutorials and API reference for the Julia implementation.

## What is here?

- **Vignettes**: rendered Julia and Python walkthroughs, including HTML and PDF downloads.
- **API reference**: exported types and functions from `June.jl`.

## Tutorials

Start with the [vignette overview](generated/vignettes/index.md), then work through the numbered tutorials from quickstart to advanced simulation setup.
