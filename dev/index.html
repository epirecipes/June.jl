June.jl

June.jl is a Julia port of JUNE, the individual-based epidemiology simulator.

This docs site publishes the package tutorials and API reference for the Julia implementation.

What is here?

  • Vignettes: rendered Julia and Python walkthroughs, including HTML and PDF downloads.
  • API reference: exported types and functions from June.jl.
